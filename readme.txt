== Description ==

A plugin to scan WordPress core directories to check the files integrity.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/wp-core-integrity` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Core Integrity->Settings screen to configure the plugin
